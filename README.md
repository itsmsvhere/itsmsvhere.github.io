## 🛠 TECH STACK

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 — clamp(), grid, flexbox, custom properties |
| Animation | GSAP 3.12 + ScrollTrigger |
| Video | HLS.js (Mux stream) |

Pure vanilla — 3 HTML files, self-contained.

### Mobile Adaptations
- Custom cursor hidden on touch devices
- Native touch scroll restored
- Paper stack responds to swipe gestures
- Scroll resistance disabled on mobile
- Safe area insets for notch devices (`viewport-fit=cover`)
- Floating panels hidden, layouts stacked
- Magic nav hidden below 1024px
